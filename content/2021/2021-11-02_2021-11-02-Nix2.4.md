私の知る限りではこれで普段

```
nix profile upgrade
```

とだけすればよくて、気が向いたら

```
nix profile wipe-history --older-than 7d
```

で古いプロファイルの回収ということらしい。flakeはTTLが切れたら勝手に更新してくれる。

検索したいなら：

```
nix search nixpkgs package-name
```

flakeはキャッシュされるので`nix-env -qa`とは比べ物にならない。


なんか成熟感と新鮮さが同時にやってきた。
